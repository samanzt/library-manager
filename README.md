# Library Manager

A simple desktop Library Management System built with Python, Tkinter, and SQLite.

## Features

* Add new books
* Update book information
* Delete books
* Search books by title, author, year, category, or status
* Borrow books
* Return books
* Automatic SQLite database creation
* User-friendly graphical interface

## Technologies Used

* Python
* Tkinter
* SQLite3

## How to Run

1. Install Python 3.
2. Download the project files.
3. Run:

```bash
python "Library Manager.py"
```

Or use the provided executable file.
